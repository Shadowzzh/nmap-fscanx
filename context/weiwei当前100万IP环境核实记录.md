## 2. 结论摘要

当前 `weiwei` 上已经存在一套新的“稀疏活主机 + 大量静默地址”实验环境，新增目标地址空间精确为 `1000000` 个 IP。

这 `1000000` 个地址不是单个连续大网段，而是由下面 `7` 个 CIDR 前缀拼接得到：

- `10.240.0.0/13` = `524288`
- `10.248.0.0/14` = `262144`
- `10.252.0.0/15` = `131072`
- `10.254.0.0/16` = `65536`
- `10.255.0.0/18` = `16384`
- `10.255.64.0/23` = `512`
- `10.255.66.0/26` = `64`

合计：

- `524288 + 262144 + 131072 + 65536 + 16384 + 512 + 64 = 1000000`

### 7.1 关于百万地址空间

可以确认，`weiwei` 里当前确实存在一套精确为 `1000000` 地址的稀疏扫描实验环境，而不是“约等于 `100` 万”。

### 7.2 关于活主机分布

当前至少已经明确有 `3` 个活 IP，分别分布在：

- `10.240.0.0/13`
